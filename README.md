## Ibasho Protocol
Данный проект представляет собой сюжетную двумерную RPG игру в пиксельной стилистике, с детективным элементом. Игра повествует о молодой некромантке, которую несправедливо ненавидит общество в связи с предрассудками.
Данный проект базируется на движке Unity.
Статус проекта: MVP; Реализовано большое количество механик внутри боевой системы, а также разработана новелльная часть.

### Основные системы
1. Боевая система реализована, главным образом, вокруг классов BattleManager (контролль хода) и LivingEntity (хранение данных). Первый ответственнен за определение боевой сцены, ограничение хода, спавн сущностей, текущее состояние боя; второй, вместе с AbilityController, реализуют исполнение навыков (атаки/способности) боевых сущностей, управляют их характеристиками.
2. Новелльная часть выполнена при помощи плагина Ink, основным классом является DialogueManager. Он отвечает за определение следующих реплик персонажей, выборов в диалоге, а также за переключение звуков и музыки диалогов 
Также присутствует множество вспомогательных классов, например, статический NextSceneSettingsStatic — ключевой класс в передаче данных между сценами. Многие элементы данных вынесены в вид ScriptableObject, что позволяет быстро создавать: новые сцены битв, базовые характеристики существ, волны врагов на уровне.
